Main Features

File and Folder Selection

  • Browse and select individual image or video files
  • Browse and select entire folders containing multiple images and videos

Model Selection

  • Choose from multiple face detection models:
    • YOLOv8n-face (Nano):
      • Fastest inference speed
      • Smallest model size (2.7MB)
      • Lower accuracy compared to larger models
      • Ideal for real-time applications or devices with limited resources
    • YOLOv8m-face (Medium):
      • Balanced performance between speed and accuracy
      • Moderate model size (27.3MB)
      • Suitable for most general-purpose face detection tasks
    • YOLOv8l-face (Large):
      • Highest accuracy among YOLOv8 models
      • Larger model size (59.2MB)
      • Slower inference speed compared to smaller models
      • Best for applications requiring high precision
    • RetinaFace:
      • Different architecture from YOLO models
      • Provides additional facial landmarks (eyes, nose, mouth)
      • Good balance of speed and accuracy
      • Suitable for tasks requiring facial feature localization
  • Automatic model download if not present on the system

Confidence Threshold Adjustment

  • Adjustable confidence threshold slider (0.0 to 1.0)
  • Default confidence values set based on selected model

Face Recognition Process

  • Support for both image and video processing
  • Batch processing of multiple files in a folder
  • Real-time progress tracking with progress bar and percentage display
  • Detailed logging of the recognition process

Results and Output

  • Automatic creation of a timestamped results folder
  • CSV output with detailed face detection information
  • Summary CSV with overall statistics for processed files
  • Visual results saved for both images and video frames
  • Automatic opening of results folder upon completion

User Interface

  • Intuitive GUI with file/folder selection, model choice, and confidence adjustment
  • Real-time status updates in the application window
  • Error handling and user notifications

Supported File Formats

  • Images:
    • JPEG (.jpg, .jpeg)
    • PNG (.png)
    • BMP (.bmp)
  • Videos:
    • MP4 (.mp4)
    • AVI (.avi)
    • MOV (.mov)
  • You can select individual files or folders containing these formats for processing
