Getting Started

Welcome to the Face Recognition Toolbox user guide! This manual will help you navigate the graphical user interface (GUI) and make the most of our face recognition software.

Installation

To install the Face Recognition Toolbox:

  1. Download the installer from our official website.
  2. Run the installer and follow the on-screen instructions.
  3. Once installed, you'll find the application in your Start menu or Applications folder.

Launching the Application

To start the Face Recognition Toolbox:

  1. Locate the application icon in your Start menu or Applications folder.
  2. Double-click the icon to launch the application.
  3. The main interface will appear, ready for use.

Interface Overview

The main interface consists of several key areas:

  • File/Folder Selection: Choose images or videos for processing.
  • Model Selection: Pick the face recognition model to use.
  • Confidence Threshold: Adjust the detection sensitivity.
  • Start Button: Begin the recognition process.
  • Results Display: View the output of the recognition process.
