Understanding Results

CSV Output

1. results.csv

1.1 Single File Mode (Image or Video)

  • filename: Name of the processed file or video frame
  • face_detected: 1 if a face was detected, 0 otherwise
  • face_count: Number of faces detected in the image or video frame
  • face_X_x, face_X_y: X and Y coordinates of the center of the face bounding box (X is the face number)
  • face_X_width, face_X_height: Width and height of the face bounding box
  • face_X_confidence: Confidence score for the detected face

1.2 Folder Mode

Same columns as Single File Mode, but includes results for all processed images and video frames in the folder

2. summary.csv

2.1 Single File Mode

  • path: Name of the processed file
  • type: "image" or "video"
  • total_processed_frames: Number of frames processed (1 for images, actual frame count for videos)
  • total_duration: Duration of the video in seconds (N/A for images)
  • processed_frames_with_faces: Number of frames where faces were detected
  • face_percentage: Percentage of frames with faces detected
  • model: Name of the face detection model used
  • confidence_threshold: Confidence threshold used for detection

2.2 Folder Mode

Same columns as Single File Mode, but includes two rows:

  1. Summary for all images in the folder
  2. Summary for all videos in the folder

The rows contain:

  • path: Name of the processed folder
  • type: "image(s)" or "video(s)"
  • total_processed_frames: Total number of images or video frames processed
  • total_duration: Total duration of all videos (N/A for images)
  • processed_frames_with_faces: Total number of images or frames where faces were detected
  • face_percentage: Overall percentage of images or frames with faces detected
  • model: Name of the face detection model used
  • confidence_threshold: Confidence threshold used for detection

These CSV files provide detailed information about the face detection results, allowing users to analyze the performance of the chosen model on their dataset.

2. Image Output

The application generates visual output for all processed images and video frames where faces are detected. These output images are saved in the results folder and include the following features:

2.1 Bounding Boxes

  • Green rectangles are drawn around each detected face
  • The size and position of these boxes correspond to the face_X_x, face_X_y, face_X_width, and face_X_height values in the results.csv file

2.2 Confidence Scores

  • The confidence score for each detected face is displayed above the corresponding bounding box
  • This score ranges from 0 to 1, with higher values indicating greater confidence in the detection
  • The displayed score corresponds to the face_X_confidence value in the results.csv file

2.3 Output File Names

  • For single image files: The output image retains the original filename
  • For video files: Each processed frame is saved as a separate image file
    • Filename format: [video_name]_[frame_number]_[timestamp].jpg
    • Example: video1_120_4.0.jpg (frame 120 at 4.0 seconds from video1)

These visual outputs provide an intuitive way to verify the face detection results and assess the performance of the chosen model on your dataset.
