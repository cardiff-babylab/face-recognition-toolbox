Support and Updates

Getting Help

If you need assistance with the Face Recognition App, there are several ways to get help:

  • Documentation: This user guide provides comprehensive information about using the application.
  • GitHub Issues: For bug reports or feature requests, please visit our GitHub Issues page.
  • GitHub Discussions: For general questions or discussions, participate in our GitHub Discussions.
  • Email Support: For direct assistance, contact our support team at support@facerecognitiontoolbox.com.

Checking for Updates

To ensure you have the latest version of the Face Recognition App:

  1. Open the application and navigate to Help > Check for Updates.
  2. If an update is available, you'll be prompted to download and install it.
  3. Alternatively, visit our GitHub Releases page to manually download the latest version.
