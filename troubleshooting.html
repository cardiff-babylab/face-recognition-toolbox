Troubleshooting

Common Issues

If you're experiencing any issues with our software, please check the following resources:

  • For bug reports or unexpected errors, please submit an issue on our GitHub page.
  • For general questions or discussions, visit our GitHub Discussions page.

Error Messages

If you encounter any error messages, please follow these steps:

  1. Check our FAQs section below for common solutions.
  2. Search our GitHub Issues to see if the problem has been reported or resolved.
  3. If you can't find a solution, please create a new issue with the following information:
    • A clear description of the problem
    • Steps to reproduce the error
    • The exact error message you received
    • Your system information (OS, browser version, etc.)

FAQs

For frequently asked questions and their answers, please visit our Q&A section on GitHub Discussions. If you can't find an answer to your question, feel free to start a new discussion thread.

We appreciate your feedback and contributions to improving our software!
